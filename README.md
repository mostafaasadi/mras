# MRas
### A bash script to manage and monitor Raspberry Pi over ssh
### It's just a personal tutorial project for *fun* :blush:
# 
# Installation 
- ```$ git clone https://github.com/mostafaasadi/mras```
- ```$ cd mras```
- ```$ chmod +x mras.sh```
- ```$ ./mras.sh```
